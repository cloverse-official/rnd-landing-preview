# RnD 사전신청 랜딩페이지

석사 이상 고학력자 소개팅 앱 **RnD**의 사전가입 신청을 받는 웹 랜딩페이지입니다.
모두 **무료 도구**(GitHub Pages + Google Sheets + Google Apps Script)로 만듭니다.

## 폴더 구조
```
rnd-landing/
├─ index.html          # 신청서 페이지 (랜딩2~3)
├─ styles.css          # 디자인 (랜딩3)
├─ script.js           # 입력 검증 + 제출 처리 (랜딩2)
├─ terms.html          # 이용약관 (랜딩3)
├─ privacy.html        # 개인정보 처리방침 (랜딩3)
├─ apps-script/
│   └─ Code.gs         # 구글 중계기 코드 (랜딩1) ← 구글에 붙여넣는 코드
├─ assets/             # 로고·OG 이미지·파비콘
└─ README.md
```

## 데이터 흐름
신청서 웹페이지 → (제출) → Google Apps Script(중계기) → Google Sheets(자동 저장)

## 랜딩1 설정 방법 — 구글 시트 연동

> 모두 무료이고, 구글 계정(지메일)만 있으면 됩니다.

### 1단계. 구글 스프레드시트 만들기
1. https://sheets.google.com 접속 → 구글 로그인
2. **빈 스프레드시트**(`+` 새로 만들기) 클릭
3. 왼쪽 위 제목을 `RnD 사전신청`으로 변경
   - ※ 시트 안에 직접 제목 줄을 입력할 필요는 없습니다. 코드가 자동으로 만들어 줍니다.

### 2단계. Apps Script 열기
- 상단 메뉴 **확장 프로그램 → Apps Script** 클릭 → 새 편집기 탭이 열립니다.

### 3단계. 코드 붙여넣기
1. 편집기에 기본으로 있는 내용(`function myFunction() {}`)을 모두 지웁니다.
2. `apps-script/Code.gs` 파일의 내용을 전부 복사해 붙여넣습니다.
3. 상단의 **저장**(💾 디스크 아이콘) 클릭.

### 4단계. 연결 테스트 (배포 전 점검)
1. 편집기 상단 함수 선택 칸에서 **`testInsert`** 선택 → **실행** 클릭
2. 처음엔 **권한 승인** 창이 뜹니다 → 본인 구글 계정 선택 →
   "Google에서 확인하지 않은 앱" 경고 시 **고급 → (프로젝트 이름)으로 이동 → 허용**
3. 스프레드시트로 돌아가 **신청목록** 탭에 샘플 한 줄(홍길동…)이 생겼으면 성공 ✅

### 5단계. 웹 앱으로 배포 (신청서가 보낼 주소 만들기)
1. 편집기 오른쪽 위 **배포 → 새 배포** 클릭
2. **유형 선택**(⚙️ 톱니바퀴) → **웹 앱** 선택
3. 설정:
   - **다음 사용자로 실행(실행 계정):** 나
   - **액세스 권한이 있는 사용자:** **모든 사용자**
4. **배포** 클릭 → **웹 앱 URL**(`.../exec`로 끝남) 복사 → 안전한 곳에 보관
   - 이 URL을 랜딩2의 신청서 코드에 연결합니다.

> 코드/설정을 바꾸면 **배포 → 배포 관리 → (연필) 수정 → 새 버전 → 배포**로 갱신하세요.

### 설정 바꾸기
- 같은 이메일 중복 신청 막기/허용: `Code.gs`의 `BLOCK_DUPLICATE_EMAIL` 값을 `true`/`false`로.
